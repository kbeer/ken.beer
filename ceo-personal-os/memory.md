# Memory — Patterns and Insights Over Time

This document is your personal knowledge base — a living record of what you've learned about yourself, your work, and how you operate.

Update this after every quarterly and annual review. The goal is to capture patterns, insights, and lessons so you don't forget what you've learned.

---

## How to Use This Document

1. **After each quarterly review:** Add new patterns, insights, or lessons learned
2. **After each annual review:** Add major insights and update recurring patterns
3. **Before setting new goals:** Read this document to remind yourself what you've learned
4. **When making big decisions:** Reference this to avoid repeating past mistakes

This is not a diary. It's a database of self-knowledge.
